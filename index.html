Set-SF-PnPDefaultFeatures

Sets the State Fund Default Features

`Set-SF-PnPDefaultFeatures` Sets the State
Fund Default Features based on the Site Template
via PnP PowerShell

Syntax 

Set-SF-PnPDefaultFeatures [-URL] <String> [<CommonParameters>]

Parameters

Name Alias Description Required? Pipeline Input Default Value
-URL URL of the site to perform action true false

Examples

EXAMPLE 1 
Set-SF-PnPDefaultFeatures -URL "https://statefund.sharepoint.com/sites/testcomm"
